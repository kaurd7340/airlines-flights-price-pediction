✈️ Airline Ticket Price Prediction & Analytics Dashboard

📌 Project Overview

The Airline Ticket Price Prediction & Analytics Dashboard is an end-to-end Data Analytics and Machine Learning project that combines Power BI for business intelligence and Streamlit for machine learning deployment.

The project analyzes airline ticket pricing patterns, flight trends, and airline performance while also providing a machine learning-powered prediction system that estimates ticket prices based on user-selected flight details.

This project demonstrates the complete data analytics workflow including data cleaning, visualization, dashboard development, machine learning model building, and deployment.

⸻

🎯 Project Objectives

* Analyze airline ticket pricing trends.
* Build interactive dashboards using Power BI.
* Develop a machine learning model for ticket price prediction.
* Deploy the prediction model through Streamlit.
* Generate actionable insights from airline data.

🤖 Machine Learning Model

Problem Type

Regression

Target Variable

Ticket Price

Features Used

* Airline
* Source
* Destination
* Cabin Class

Algorithm

Random Forest Regressor

Workflow

Data Preprocessing

* Cleaned and transformed the airline dataset.
* Selected relevant features for prediction.
* Applied Label Encoding to categorical variables.
* Prepared the dataset for machine learning training.

Model Training

* Split the dataset into training and testing sets.
* Trained the model using Random Forest Regressor.
* Evaluated model performance using R² Score and prediction accuracy.

Model Deployment

* Saved the trained model using Joblib.
* Integrated the model with Streamlit.
* Built an interactive user interface for real-time predictions.
